License Plate Recognition Test

How to test ?

  1. Load images from the given directory

    2. Only images with following suffixes are allowed:

    • jpg, JPG, png, PNG
  2. Load text file, which contains original license plate numbers

    3. Format of text file

    • image name, original license plate numbers

    Example:


    image_1.jpg PO123KL

    test8.png KK358LL


    Image name must contains suffix!


  3. Select the adapter you want to use for testing

    4. You can choose from these adapters:

    • Javaanpr adapter
    • PlateRecognizer adapter
    • OpenCV 3 License Plate Recognition adapter (RecognitionCpp adapter)
  4. Start analysis

    5. The results will be displayed below


  5. Then you can save the recognition results to the given text file

    6. Text file contains:

    • path to the directory
    • image name, recognized characters, percentage
    • total number of recognized plates with percentage
    • total number of recognized characters with percentage

About adapters

  1. Javaanpr

    2. Location of config.xml: >> Project folder >>

    For working with this library, you must set paths of following entries in config.xml

    • char_learnAlphabetPath
    • char_neuralNetworkPath
    • intelligence_syntaxDescriptionFile

    You can set properties in the config.xml file located in the resources directory, such as:

    • intelligence_classification_method
    • thresholding mode
    • and more...
  2. PlateRecognizer
  3. OpenCV 3 License Plate Recognition adapter (RecognitionCpp adapter)

    4. You must set the PATH system variable to the lib directory in the project
